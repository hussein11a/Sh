# SH Website

موقع ويب متطور مبني باستخدام تقنيات حديثة.

## المميزات
- تصميم حديث ومتجاوب
- أداء عالي
- سهولة التخصيص

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
- Node.js

## البدء
1. نسخ المستودع
```bash
git clone https://github.com/[username]/SH.git
```
2. تثبيت الاعتمادات
```bash
npm install
```
3. تشغيل الموقع محلياً
```bash
npm run dev
```
